# Coding Ninjas Website Clone

A fully responsive, high-performance web clone of the modern Coding Ninjas website built with **React** and **Tailwind CSS**. This project faithfully replicates the vibrant, dark AI-themed aesthetic and lead-generation flow of the original site.

![TailwindCSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white)
![Vite](https://img.shields.io/badge/Vite-B73BFE?style=for-the-badge&logo=vite&logoColor=FFD62E)

## 🚀 Key Features

- **Modern Hero Section**: A stunning dark-themed *AI Advantage* hero area with an interactive lead-generation form.
- **Glassmorphism UI**: Beautifully styled components and dropdown navigations built completely utilizing Tailwind CSS utility classes.
- **Dynamic Portfolios & Courses**: Interactive tabs mapping structure for curriculums (Data Science, DSA, Full Stack Web Development).
- **Hiring Partners & Testimonials**: Customized grid layout demonstrating success stories from top-tier tech companies like Google, Microsoft, and Amazon.
- **Hover Micro-animations**: Smooth, interactive feel with gradient shifts, text highlights, and scale transformations.
- **Fully Responsive Structure**: Designed mobile-first, ensuring an optimal viewing and browsing experience across all devices.

## 🛠️ Technology Stack

- **Framework**: React (Bootstrapped with Vite)
- **Styling Engine**: Tailwind CSS v4
- **Icons Toolkit**: Lucide React
- **Routing**: React Router DOM *(Prepared for multi-page integration)*

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone <your-repository-url>
   ```

2. **Navigate into the directory:**
   ```bash
   cd SLAB-II
   ```

3. **Install all dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Visit the app:**
   Open your browser and navigate to `http://localhost:5173`.
